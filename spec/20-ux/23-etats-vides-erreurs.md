# /spec/20-ux/23-etats-vides-erreurs.md

## États vides
- Aucun tournoi: CTA "Créer un tournoi" + "Essayer la démo"
- Tour sans tables: CTA "Créer une table" + rappel "gabarit auto possible"
- Aucun score validé: classement affiche 0 partout, message "En attente des premiers scores"

## Erreurs / validations
- Scores non valides: message court + highlight champ
- Joueur déjà assigné sur le tour: blocage + info "déjà à la table X"
- Lien joueur expiré: écran dédié + info "demander réactivation à l’admin"
