# /spec/30-brand/32-composants-shadcn-guidelines.md

## Principes de composants
- Réutiliser ShadcnUI autant que possible
- Components de base attendus:
  - Button, Card, Badge, Tabs
  - Dialog, Drawer (mobile)
  - Toast
  - Input, Select, Combobox (recherche joueur)
  - Accordion (détails tour / tables)
  - Skeleton (loading)

## Design system
- Variants cohérents (primary = accent #51bdcb)
- États: default / hover / pressed / disabled / loading
