# /spec/50-data-rules/53-validation-correction-cloture.md

## Validation
- Une table validée déclenche recalcul classement et devient "validée".

## Correction (toujours possible par admin)
- L’admin peut déverrouiller, modifier, revalider.
- Avertissement UX: "cela impacte le classement".

## Clôture
- Tournoi clôturé:
  - classement final figé (mais correction autorisée si admin le souhaite)
  - historique conservé
  - exports disponibles
